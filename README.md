# **LittleBusiness**

**LittleBusiness** es una plataforma web diseñada para que pequeños negocios puedan tener un perfil personalizado donde sus clientes puedan explorar productos, hacer reservas de servicios y realizar compras de manera fácil y directa. Cada negocio cuenta con su propia URL dentro de la plataforma, lo que facilita a los clientes el acceso directo a su perfil y su catálogo de servicios y productos.

### **Tecnologías Utilizadas**

- **Frontend**: Blade (HTML, CSS, JavaScript)
- **Backend**: Laravel v11 (PHP)
- **Base de Datos**: MySQL
- **Control de Versiones**: GitHub
- **Entorno de Desarrollo**: Docker
